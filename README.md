# Embedded Systems Projects

This repository contains hands-on **Embedded Systems projects** developed as part of my coursework and self-directed learning. The projects focus on **microcontroller programming**, **hardware–software integration**, and **real-world system behavior** using industry-relevant tools and practices.

## Overview

The goal of this repository is to demonstrate:
- Practical application of embedded systems concepts
- Low-level programming and hardware control
- System design, testing, and debugging
- Real-life embedded system use cases

## Technologies & Tools

- **Microcontrollers:** Arduino, ATmega-based boards, ESP32
- **Programming Languages:** C / C++
- **Development Tools:** Arduino IDE, PlatformIO
- **Hardware Components:** Sensors, actuators, LEDs, motors, relays, push buttons, resistors, breadboards
- **Communication Protocols:** UART, I2C, SPI
